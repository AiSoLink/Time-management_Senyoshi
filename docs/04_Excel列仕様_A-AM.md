- A:運行ID / B:運行日 / C:乗務員ID / D:乗務員名 / E:車両ID / F:車両番号 / G:所属営業所 / H:出庫メーター / I:帰庫メーター / J:総走行距離 / K:出庫日時 / L:帰庫日時 / M:拘束時間_分割前 / N:拘束時間_分割後 / O:労働時間 / P:労働時間_昼_分割前 / Q:労働時間_夜_分割前 / R:労働時間_昼_分割後 / S:労働時間_夜_分割後 / T:運転時間 / U:待機時間 / V:休憩時間_分割前 / W:休憩時間_昼_分割前 / X:休憩時間_夜_分割前 / Y:休憩時間_分割後 / Z:休憩時間_昼_分割後 / AA:休憩時間_夜_分割後 / AB:休息時間 / AC:休息時間_昼 / AD:休息時間_夜 / AE:休息時間採用 / AF:運行間休息 / AG:ランキング / AH:分割開始1 / AI:分割終了1 / AJ:分割1_作業時間_分 / AK:分割開始2 / AL:分割終了2 / AM:分割2_作業時間_分

## フェーズ1（PDF→Excel）で埋める/空欄

### 埋める
- メーター差：総走行距離
- 拘束時間_分割前（分）
- 労働時間_昼_分割前 / 労働時間_夜_分割前（分）※拘束区間を昼夜配賦
- 運転時間（分）：到着−直前出発の合算（初期prev_depart=出庫出発）
- 待機時間（分）
- 休憩（分割前：合計/昼/夜）
- 休息（合計/昼/夜）＋上位2回の開始/終了/分
- 得点（安全点：機種差分）

### 空欄（後段Excelマクロ）
- “分割後”列すべて
- 休息時間採用 / 運行間休息 / ランキング
